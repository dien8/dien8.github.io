Data Handling on this Website


Description and Scope of Data Processing

The server of this website logs at least the following data per request:

  • Date and time of day of the request
  • IP address the request originated from
  • Requested webpage (URL)

Additionally, your browser potentially sends some data which is automatically stored in the web server log files. Typically, this includes:

  • Web browser type and version
  • Operating system
  • Referrer URL (the previously visited website)

This data can only be linked to individual persons in exceptional cases. No techniques to circumvent the effectivity of anonymising software are used.

Legal Basis of Data Processing

Legal basis for the temporary storage of this data and logfiles is Art. 6 Abs. 1 lit. f DSGVO.

Purpose of Data Processing

Data is stored in logfiles to ensure the operation of the website. Additionally, it is used to optimize our website and for security of our IT systems. No analysis of this data for marketing purposes is conducted.

These purposes justify our interest in processing this data according to Art. 6 Abs. 1 lit. f DSGVO.

Duration of Storage

Stored data is deleted if the purpose of its storage ceases to exist. In case of storage of your data for providing this website to you, this occurs when the respective session is closed.

In case of storage of your data to ensure the security of our IT systems, this will occur after no more than seven days. Storage longer than that is possible. In that case, IP addresses of users will be deleted or masked, so that no further correlation to the client is possible.

Methods to Appeal or Delete

Collection of this data for providing this website and storage in logfiles is mandatory for the secure operation of this website. Thus, no method to appeal is available.
